Dative (under development)

  • Research-oriented: The Dative app is designed to help researchers fully utilize their linguistic data and facilitate their research.
  • Powerful Search: Search expressions targeting specific data fields and using regular expressions can be conjoined, disjoined, or negated to produce powerful queries.
  • Input Validation: Users can specify valid characters, or character sequences, for each field. The app will warn users when there are invalid inputs.
  • Morphological cross-referencing: The system indicates whether newly entered morpheme-gloss pairs have matching entries in the existing data, minimizing instances of misspelling and inconsistent glossing.
  • Orthography conversion: When the object language doesn't have a unified orthographic system, orthography conversion allows users to use their preferred orthographic conventions, while still creating a data set with a single underlying orthography.
  • Export: Individual entries, search results, texts, entire corpora, and sub-corpora are exportable to variety of formats including (Xe)LaTex, XML, CSV, JSON, TSV, and plain text.
  • Text Creation: Users can extract data points and create texts without leaving the Dative app. By exporting these texts to other formats, users can quickly convert their raw data into drafts of research papers and text collections.
  • Import: Import from CSV, TSV, and plain text will be available.
  • Grammar Modelling: The Dative app allows users to build computational models of phonologies and morphologies based on stored data. From these models, users can create morphological parsers to facilitate morphological analysis as well as to automate the testing of grammatical models against large data sets.
  • Please note:
    • You can open the same corpus in both Dative and Spreadsheet. Dative is designed to help researchers thoroughly examine their linguistic data sets. For quick data entry, you can use Spreadsheet.
    • In addition to the databases created with LingSync, Dative can also interface with the databases created with the Online Linguisic Database (OLD).
