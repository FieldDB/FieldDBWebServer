Dative (under development)

  • One Database for Both Apps: One and the same database underlies both Dative and Spreadsheet apps, thus you can access your data using either app. Dative is designed to help researchers to thoroughly examine their linguistic data sets. For quick data entry, you can use the Spreadsheet app.
  • Research-oriented
  • Powerful Search
  • Data Consistency
  • Import
  • Export
  • Grammar Modelling
  • Research Paper Creation
